# User Login Template

## Config  

Install dependencies : 
```
$ npm install
```
Config database in "app/config/db.config.js"  
Modify secret key in "app/config/auth.config.js"

To launch the API : 
```
$ node server.js 
```


## Requirement

Node 14

MySQL

## More Information

While launching the server, the DB is created if it not already done.
